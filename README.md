# React marathon — логи и сценарии

**Репозиторий:** [github.com/nixjke/react-log](https://github.com/nixjke/react-log)

Учебные задания лежат в **`src/react/`** — см. [оглавление и порядок работы](src/react/README.md).

**Техническое описание** (теория React, фазы, эффекты, очередь задач, разбор по сценариям): [docs/TECHNICAL.md](docs/TECHNICAL.md).

## Запуск

```bash
npm install
npm run dev
```

Откройте адрес из терминала (обычно http://localhost:5173/), выберите номер сценария и проверьте вывод в **консоли браузера** (F12 → Console).

## Сборка

```bash
npm run build
```
