# Марафон: разбор по файлам

Теория React, фазы рендера/эффектов, разбор сценариев: [docs/TECHNICAL.md](../docs/TECHNICAL.md) · репозиторий: [github.com/nixjke/react-log](https://github.com/nixjke/react-log).

Как работать:

1. Откройте нужный **`.tsx`** ниже — там код, комментарии и **вопросы для самопроверки**.
2. Запишите ответы устно или в конспекте.
3. Запустите проект (`npm run dev`), откройте сценарий с тем же номером в меню и **консоль браузера** (F12 → Console), чтобы сверить поведение с разбором.

| № | Файл | Тема |
|---|------|------|
| 1 | `1__log-order.tsx` | Порядок рендера и эффектов (`useEffect` / `useLayoutEffect`) |
| 2 | `2__log-order.tsx` | Порядок в дереве компонентов (Foo → Bar → Baz) |
| 3 | `3__memo-callback.tsx` | `useCallback`, `memo` |
| 4 | `4__rerender.tsx` | Ререндер, `children`, типичные ошибки в разметке |
| 5 | `5__keys.tsx` | `key` в списках, `render` / `effect` в консоли |
| 6 | `6__context.tsx` | Context, обновление значения |
| 7 | `7__jsx.tsx` | JSX: вложенность, массивы в разметке |
| 8 | `8__suspense.tsx` | Suspense, задание с `TODO` в `Bar` |

Сценарий **8**: в файле остаётся задание; рабочий запуск для демонстрации Suspense вынесен в `App.tsx` (после своей реализации можно подключить свой вариант).
